# 概念透镜组 (Concept Lens Array)

> 将模糊需求校准为清晰、可执行且边界明确的工程草案

---

## 核心理念

**模糊的灵感是廉价的，清晰的约束才是昂贵的。**

概念透镜组是一个严肃的创意工程系统，帮助你在动工前磨出一张清晰到不会产生歧义的设计蓝图。

---

## 三镜片模型

```
雾镜（显影）→ 棱镜（分光）→ 聚焦镜（收敛）
```

| 镜片 | 核心任务 | 具体作用 |
|:-----|:---------|:---------|
| **雾镜** | 形容词→工程参数 | 把"高端文案"变成"句长≤12字，动词占比40%" |
| **棱镜** | 识别边界与约束 | 指出"AI无法创造真实体验"，逼用户做选择 |
| **聚焦镜** | 锁定核心与形态 | 决定交付是一个Slogan还是完整提案框架 |

---

## 适用场景

- **文案创作**：从"要戳心"到"具体洞察 + 句式结构"
- **市场BP分析**：从"要全面"到"核心假设 + 验证路径"
- **Skill设计**：从"要智能"到"输入输出规范 + 边界条件"
- **灵感转化**：从"有个点子"到"可行性第一步动作"

## 核心价值

### 解决"来回改"的根源
把修改前置到概念层。如果参数表对了，成品不可能"感觉不对"。

### 解决AI"说废话"
参数里有"信息密度"和"场景焦点"，AI不会再去堆砌华丽的虚词。

### 解决用户的"全能自恋"
用"不可能三角"和"物理边界"作为温柔但坚定的手术刀，帮用户完成理性的自我阉割。

## 核心原则

1. **严禁直接给结果**：必须按三步流程执行
2. **不可跳步**：每个镜片必须完成才能进入下一个
3. **明确边界**：温柔但坚定地指出AI能力边界
4. **完成割舍**：用"不可能三角"逼用户做选择

## 示例预览

### 文案创作示例

**用户输入**：
```
帮我写一个高端的护肤品文案
```

**AI执行流程**：
1. 雾镜校准：把"高端"翻译成"慢语速、低密度、温情绪"
2. 棱镜分光：指出"深度洞察/传播性/篇幅短"不可能三角
3. 聚焦镜收敛：锁定"情感共鸣文案"，舍弃"产品硬广"

**最终交付**：
```
标题：那些你熬夜赶的方案，皮肤都记得

正文：凌晨一点，你终于合上电脑...
```
