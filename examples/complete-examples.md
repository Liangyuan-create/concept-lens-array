本文档包含概念透镜组的完整使用示例，涵盖四个核心场景。

## 示例一：文案创作场景

### 用户输入

```
帮我写一个高端的护肤品文案
```

### AI 执行流程

#### 第一步：雾镜校准

**AI 输出**：

```
为了避免我们陷入"感觉不对"的死循环，请先暂停对最终成品的想象。我们需要先把你脑中的"高端"翻译成可测量的工程参数。

请回答以下至少 2 个问题：

1. 如果要把"高端"拍成一张照片，背景里有什么？绝对不能出现什么？
2. 这个文案最核心的阅读场景是怎样的？是在专柜试用时看？还是刷朋友圈时扫一眼？
3. 你见过最糟糕的护肤品文案是什么？它为什么让你觉得"不高端"？
```

**用户回答**：

```
1. 背景是干净的白色或米色，有自然光，绝对不能出现促销标签、夸张的"买一送一"
2. 应该是在晚上睡前刷手机时看到，那时候比较放松
3. 最讨厌那种"震惊！三天见效！"的夸张标题，感觉很low
```

**AI 输出《概念翻译草案》**：

```markdown
# 概念翻译草案

## 一、拟用调性参数

| 参数维度 | 参数值 | 说明 |
|:---------|:-------|:-----|
| 语速 | 慢 | 睡前放松场景，需要舒缓节奏 |
| 信息密度 | 低 | 避免信息过载，符合放松状态 |
| 情绪温度 | 温 | 不冷不热，温和治愈感 |
| 句式风格 | 中长句 | 适合睡前阅读，有节奏感 |
| 词汇层级 | 通俗+少量专业 | 让用户能理解，但保持专业感 |

## 二、场景约束

**核心阅读场景**：晚上睡前刷手机

**用户注意力焦点**：放松、治愈、期待明天更好的自己

**场景限制因素**：
- 时间限制：3-5秒抓住注意力，15-30秒完整阅读
- 环境干扰：中（可能有其他App通知）
- 认知负荷：低（睡前不想动脑）

## 三、禁用清单

| 序号 | 禁用类型 | 具体内容 | 原因 |
|:-----|:---------|:---------|:-----|
| 1 | 禁词 | "震惊"、"三天见效"、"奇迹" | 夸张营销感，不高端 |
| 2 | 禁风格 | 促销风格、标题党 | 与高端感冲突 |
| 3 | 禁元素 | 促销标签、买一送一、限时优惠 | 破坏信任感 |

---

*草案状态：待棱镜分光验证*
```
